Personal Profile

I am Kashish Jaiswal, a dedicated and ambitious developer with expertise in front-end technologies like HTML, CSS, JavaScript, and React, alongside backend experience in Django and Strapi. With a solid foundation in databases such as SQL and PostgreSQL. I have successfully delivered diverse projects that bridge technology and real-world solutions. My portfolio includes building a dynamic event calendar application, developing an loT-enabled pollution meter to monitor air quality using Arduino, and creating a fully functional e-commerce platform using React, Strapi, and Stripe. Having completed the Metacrafters summer training program in blockchain technologies like Ethereum, Avalanche, and Polygon, I am continually expanding my skill set.
